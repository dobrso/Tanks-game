# Battle Tanks
Top-Down PvP игра, где в управление игроку дается собственный танк. \
Каждая игровая сессия представляет собой "рубилово" между игроками. \
Игровая сессия завершается, когда не остается игроков в комнате.

## Требования
* Python 3.12

## Инструкция

1. Для начала склонируйте репозиторий с игрой \
``
git clone https://github.com/dobrso/Tanks-game.git
`` 
2. Создайте виртуальное окружение \
``
py -m venv venv
``
3. Запустите виртуальное окружение \
``
venv/Scripts/Activate.ps1 (Если не сработает, попробуйте другие скрипты для активации)
``
4. Установите все зависимости из requirements.txt \
``
pip install requirements.txt
``
5. Запустите сервер и клиента (Starters)

P.S. Чтобы поиграть с другом, нужна программа, имитирующая локальную сеть (Hamachi, Radmin VPN, ...). 
Создаете локальную сеть, подключаетесь и вместо HOST ставите IP адрес хоста сервера.
